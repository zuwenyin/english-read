# AGENTS.md

> 本文件约束所有参与本项目的 AI Agent 行为。Agent 每次工作前必须阅读并遵守。

---

## 一、核心规则（最高优先级）

**遇到不清楚的问题，必须向用户询问，禁止自主决定。**

- 需求不明确或有歧义 → 向用户澄清，禁止猜测或"善意推测"
- 技术方案有多个可行选项 → 列出选项请用户决策，禁止自行拍板
- 用户指令某部分不理解 → 先澄清再执行，禁止按自己理解直接做
- 发现代码/文档与需求不符 → 告知用户，请确认如何处理

---

## 二、技术决策约束

以下选型以 `prod/technical-design.md` 为准，**不得擅自更改**：

| 决策项     | 选定方案                          |
| ---------- | --------------------------------- |
| 包管理器   | pnpm                              |
| Monorepo   | pnpm workspaces（不用 Turborepo） |
| 前端框架   | React 18 + TypeScript             |
| 构建工具   | Vite 5                            |
| UI 组件库  | Ant Design 5（通过 wrapper 封装） |
| 后端框架   | Express + TypeScript              |
| 初期数据库 | SQLite（node:sqlite 内置）        |
| 状态管理   | React Context + useReducer        |

更改上述任何决策前，必须先征得用户明确同意。

---

## 三、禁止行为

- ❌ 擅自决定技术方案或实现方式
- ❌ 擅自更改已确认的技术选型
- ❌ 对需求进行"合理推测"后执行而不确认
- ❌ 擅自引入新依赖（小版本更新除外，但需告知用户）
- ❌ 擅自删除或大幅重构用户未要求修改的代码

---

## 四、沟通规范

- 清晰陈述问题，避免模糊表述
- 提供选项时，给出每个选项的简要说明
- 预计执行结果可能与用户预期不符时，提前说明并确认

---

**一句话原则：不懂就问，不猜不做，用户确认后再执行。**
